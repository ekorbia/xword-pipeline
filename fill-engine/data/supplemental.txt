# Supplemental crossword entries that augment xwordlist.dict.

# Format: WORD;SCORE (0-100), one per line. Same format as the main dict.
# `#` comments and blank lines are ignored. Letter normalization matches
# the main dict (case-insensitive, non-letters stripped — so "Travis Kelce"
# becomes TRAVISKELCE).
#
# Loader behavior:
#   - Entries here take PRECEDENCE over duplicates in xwordlist.dict, so this
#     file doubles as a score-override mechanism for upstream entries you
#     think were mis-scored.
#   - blocklist.txt still wins — a word in both blocklist.txt and this file
#     is still excluded.
#
# Score guide (matches the main dict's conventions):
#   75-90  lively, real, NYT-grade
#   60-70  real and current, slightly niche
#   50-59  specific brand/jargon — placeable but flag with caution
#   < 50   acts as "iffy" fill; filtered out by `--max-iffy 0`
#
# Note on length for THEMED puzzles: in a 15-wide grid, theme answers are
# placed as full across entries, and a 12-letter answer can't fit with the
# required symmetric edge gaps. So 12-letter entries below (e.g. LAMARJACKSON,
# CAITLINCLARK, BARBENHEIMER) are still placeable as crossings/fill in
# non-themed puzzles but cannot serve as theme answers.
#
# Grow this list as you encounter answers that "should be in the dict but
